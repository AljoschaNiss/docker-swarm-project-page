Eine moderne Architektur mit mehreren Services

Projektübersicht

Dieses Projekt demonstriert eine Microservice-Architektur mit Docker Swarm, Reverse Proxy, Load Balancing, CI/CD und mehr.

Ziel ist es, im Heimnetz mehrere Services anbieten zu können, welche Aufgaben im täglichen Leben übernehmen können.
